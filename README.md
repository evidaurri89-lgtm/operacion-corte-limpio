# 🔍 Operación Corte Limpio  
## Corporate Intelligence & Forensic Data Investigation

> Simulación de una investigación de inteligencia operativa en una empresa familiar del sector retail con $380M MXN de facturación anual.

---

## 🎯 Objetivo del Proyecto

Desarrollar una investigación forense de datos para identificar:

- Irregularidades en la cadena de suministro  
- Erosión de márgenes operativos  
- Posibles anomalías en inventarios y merma  
- Riesgos estructurales en controles internos  

---
