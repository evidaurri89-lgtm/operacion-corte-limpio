# 🔍 Operación Corte Limpio  
## Corporate Intelligence & Forensic Data Investigation

---

## 🧠 El Problema

Una empresa familiar del sector retail con $380M MXN en facturación anual enfrenta una erosión sostenida en su margen operativo.

Aunque los ingresos han crecido, la rentabilidad ha disminuido de forma constante.

¿Ineficiencia operativa?
¿Problemas de control interno?
¿Irregularidades en la cadena de suministro?

Este proyecto simula una investigación de inteligencia operativa para responder esas preguntas utilizando análisis forense de datos.

---

## 🎯 Objetivo

Identificar y cuantificar posibles irregularidades en:

- Cadena de suministro
- Control de inventarios y merma
- Punto de venta
- Gastos operativos
- Flujos físicos vs documentales

---

## 🧭 Enfoque Metodológico

La investigación se divide en cinco líneas estratégicas:

1. Integridad de la cadena de suministro  
2. Control de inventarios y merma  
3. Análisis de punto de venta  
4. Evaluación de gastos operativos  
5. Reconciliación de flujos físicos  

Cada línea combina:

- Análisis estadístico
- Detección de anomalías
- Modelado de riesgo
- Visualización estratégica
